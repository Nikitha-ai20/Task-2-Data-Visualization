# Superstore Sales Dashboard

## Objective
Create an interactive sales dashboard using Power BI to analyze Superstore sales data.

## Dataset
- Superstore Final Dataset

## Tool Used
- Microsoft Power BI

## Dashboard Features
- Total Sales
- Total Orders
- Total Customers
- Total Products
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Sales by Segment
- Top 10 States by Sales
- Sales by Sub-Category

## Files Included
- superstore_final_dataset.csv
- Dashboard.pdf
- Dashboard.png

## Conclusion
This dashboard provides insights into sales performance, customer segments, product categories, and regional sales trends using interactive visualizations.
